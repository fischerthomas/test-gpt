
# Planning App (Vite + React + TypeScript + Tailwind)

## Développement local
```bash
npm install
npm run dev
```

## Build
```bash
npm run build
```
Les fichiers de production seront dans `dist/` (pour Vercel: Output Directory = `dist`).

## Dépendances clés
- dnd-kit (drag & drop)
- html2canvas + jsPDF (export PDF)
- lucide-react (icônes)
